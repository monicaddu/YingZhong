Index design process

Inspiration

This website called Read Me made with ReadyMag was inspiring to me during the stage of looking for references. I enjoyed the overall exprience when going through the content. Unlike other websites, it only has one page but each section has very distinct styles and the various animations involved are very engaging. The flow between each part also feels quite smooth.

Wireframes

Inspired by "Read Me", I designed different layouts for each part of the index. And I planned to place them within one web page. So the viewer would browser the index by scrolling down. On the first part, I created a content table so that each section could be linked. And for other parts, I wanted to make preview windows for all exercises and acts which lead to external pages for those who are interested.
